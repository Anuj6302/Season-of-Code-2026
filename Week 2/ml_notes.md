# Machine Learning Basics - Theory Notes


## What is Machine Learning?

Machine Learning allows computers to learn patterns from data and make predictions without being explicitly programmed.


## Types of Machine Learning

1. Supervised Learning
2. Unsupervised Learning
3. Reinforcement Learning


## Linear Regression

Used for predicting continuous values.

Example:
- Price prediction
- Temperature prediction


## Logistic Regression

Used for classification tasks.

Example:
- Spam detection


## Decision Trees

Models that make decisions by splitting data based on features.


## Ensemble Learning

Combining multiple models to improve accuracy.

Example:
Random Forest


## Bagging

Creates multiple models using different subsets of data and combines their outputs.
