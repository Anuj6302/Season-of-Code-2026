# Intro to Machine Learning — Theory Notes 

*Based on the Simplilearn "What Is Machine Learning?" intro video.*

## What is Machine Learning?
Machine Learning (ML) is a subset of Artificial Intelligence where computers learn patterns from **data** instead of being explicitly programmed with hardcoded rules. Instead of writing step-by-step instructions for every scenario, we feed the system examples, and it learns to make predictions or decisions on its own.

**Traditional Programming:**
`Rules + Data → Output`

**Machine Learning:**
`Data + Output → Rules (Model)`

## Why Machine Learning?
- Some problems are too complex for fixed rule-based logic (e.g., recognizing handwriting, detecting spam, recommending movies).
- ML models can improve over time as they see more data.
- Useful when patterns exist in data but aren't obvious or easy to code by hand.

## Types of Machine Learning
1. **Supervised Learning**
   - The model learns from labeled data (input → known output).
   - Examples: predicting house prices, spam detection, image classification.

2. **Unsupervised Learning**
   - The model finds patterns in data without labeled outputs.
   - Examples: customer segmentation, anomaly detection.

3. **Reinforcement Learning**
   - The model learns by interacting with an environment and receiving rewards or penalties.
   - Examples: game-playing AI, robotics.

## Real-World Examples
-  Email spam filters
-  Movie/show recommendation systems (Netflix, YouTube)
-  Product recommendations (Amazon)
-  Voice assistants (Siri, Alexa)
-  Self-driving car perception systems
